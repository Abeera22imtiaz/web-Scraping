# Web Scraping & Selenium Automation Project

## Overview
This project demonstrates web scraping and browser automation using **BeautifulSoup**, **Requests**, and **Selenium**. It covers extracting text, links, images, videos, and e-commerce data, as well as automating Google search and capturing screenshots.

## Features
- Extract text, links, images, and video sources from webpages  
- Scrape structured data from e-commerce websites  
- Automate Google search using Selenium  
- Capture webpage screenshots  

## Dependencies
- Python  
- `beautifulsoup4`  
- `requests`  
- `selenium`  

## How to Run
1. Install the packages using pip:  
   ```bash
   pip install beautifulsoup4 requests selenium

2. Run the Python scripts:
```bash
python your_script.py
```
## Notes
For dynamic websites, **Selenium** is used to interact with elements.

`time.sleep()` is used for simple waits, but **Explicit Waits** (`WebDriverWait`) are recommended for production code.
